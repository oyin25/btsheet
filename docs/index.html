btsheet-hosted Library

A simple and customizable bottom sheet dialog library for the web.

Installation Basic Usage API Reference Examples

Installation

To use btsheet-hosted in your project, include the following script and stylesheet from the CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/oyin25/btsheet-hosted-lib@latest/btsheet-hosted.css">
<script src="https://cdn.jsdelivr.net/gh/oyin25/btsheet-hosted-lib@latest/btsheet-hosted.js"></script>

Basic Usage

Here is a simple example of how to create and display a bottom sheet dialog using btsheet-hosted:

btsheet.hosted({
  title: "Multitap!",
  text: "Increase amount of TAP you can earn per one tap.",
  buttonText: "Get it!",
  btButton: function() {
    alert('Button Clicked!');
  }
});

API Reference

btsheet.hosted(options)

The options object allows you to customize the behavior and appearance of the bottom sheet. Below are the available options:

  • title (string): Title of the bottom sheet.
  • text (string): Description or text inside the bottom sheet.
  • buttonText (string): Text to display inside the button.
  • btButton (function): Custom function for button click behavior.
  • btImage (object): Image properties:
    • src (string): URL of the image.
    • width (string): Width of the image (e.g., '200px').
    • height (string): Height of the image (e.g., '200px').
  • btBody (object): Body styles:
    • backgroundColor (string): Background color of the sheet (e.g., 'rgba(0, 0, 255, 0.6)').
    • textColor (string): Color of the text inside the sheet.
    • titleColor (string): Color of the title inside the sheet.
    • glassEffect (boolean): Enables glass effect when true.
  • btDim (string): Dims the background with a percentage (e.g., '60%').
  • outsideTouch (boolean): Closes the sheet when clicked outside if true.
  • btInteract (boolean): Disable the interaction on the webpage when btshet is opened if false.
  • btsheet.closed(); It use to close the btsheet programatically.

Examples

Here are a few examples demonstrating the use of btsheet-hosted with different options:

Example 1: Glass Effect with Custom Button

btsheet.hosted({
  title: "Multitap!",
  text: "Increase amount of TAP you can earn per one tap.",
  buttonText: "Get it!",
  btButton: function() {
    alert('Button Clicked!');
  },
  btBody: {
    backgroundColor: "rgba(0, 0, 255, 0.6)",
    textColor: "#fff",
    glassEffect: true
  },
  btDim: "50%"
});

Example 2: Image and Custom Button Styles

btsheet.hosted({
  title: "Awesome Image!",
  text: "Here's an image example.",
  buttonText: "OK",
  btButton: function() {
    alert('Button Clicked!');
  },
  btImage: {
    src: "https://unsplash.it/400/200",
    width: "400px",
    height: "200px"
  },
  btBody: {
    backgroundColor: "#fff",
    textColor: "#333"
  },
  btDim: "70%"
});

Example 3: Use Complex with HTML Tags

btsheet.hosted({
        title: "Form Example",
        text: 
          
Fill the Form

          
          
        ,
        buttonText: "Submit",
        btButton: function() {
          const username = document.getElementById('username').value;
          const email = document.getElementById('email').value;
          if (username && email) {
            btsheet.closed();
            alert(Username: ${username}, Email: ${email});
          } else {
            alert('Please enter both username and email.');
          }
        },
        btInteract: false,  // Disables interaction with the webpage
        btDim: "50%",
        onClose: function() {
          console.log("Bottom sheet closed via X button");
        },
        outsideTouch: true
      });

Example 4: Light Complex Custom Button Styles

btsheet.hosted({
        title: "Multitap!",
        text: "Increase amount of TAP you can earn per one tap. +1 per tap for each level.",
        buttonText: "Get it!",
        button: {
          color: "#ff4757",    // Button background color
          textColor: "#fff",   // Button text color
          width: "250px",      // Custom width
          height: "40px"       // Custom height
        },
        btButton: function () {
          console.log("Button clicked!");  // Custom action on button click
          btsheet.closed();
        },
        btImage: {
          src: "https://unsplash.it/400/200",
          alt: "Custom image",
          width: "400px",
          height: "200px"
        },
        btBody: {
          textColor: "#0e0e0e",          // Customize text color inside the sheet
          backgroundColor: "#fff", // Customize background color (optional)
          titleColor: "#ff4757",      // Customize title color
          glassEffect: true           // Enable glass effect
        },
        btDim: "60%",  // Dim the background 60%
        outsideTouch: true,
        onClose: function () {
          console.log("Bottom sheet closed");
        }
      });
